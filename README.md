# refund

**refund** é um projeto front-end para solicitação e gerenciamento de reembolsos, desenvolvido utilizando HTML, CSS e JavaScript.

## Índice

- [Visão Geral](#visão-geral)
- [Tecnologias Utilizadas](#tecnologias-utilizadas)

## Visão Geral

O **refund** foi desenvolvido com o objetivo de simular um sistema de solicitação de reembolso. A aplicação permite adicionar despesas informando o nome, a categoria e o valor, exibindo as solicitações em uma lista com o total acumulado.

Utilizei HTML para estruturar a interface, CSS para estilização e organização do layout, e JavaScript para manipulação do DOM, controle de eventos e atualização dinâmica dos valores.

## Tecnologias Utilizadas

- **HTML**: Linguagem de marcação utilizada para estruturar o conteúdo.
- **CSS**: Linguagem de estilização utilizada para definir o layout e o design da aplicação.
- **JavaScript**: Linguagem utilizada para manipulação do DOM e interatividade da aplicação.
